# Momentum Website

A marketing website for the Momentum movement tracking app. This site explains the app's features, research mission, and data privacy practices.

## Structure

- `index.html` - Main website with all sections
- `styles.css` - Complete styling with dark theme
- `script.js` - Smooth scrolling and animations

## Sections

1. **Hero** - Compelling introduction to Momentum's mission
2. **What is Momentum** - Feature overview with 6 key capabilities
3. **How It Works** - 5-step process from onboarding to long-term tracking
4. **The Research Side** - Explanation of research mission and data practices
5. **Who Is This For** - 6 audience segments
6. **Your Data Is Yours** - Privacy and transparency commitments
7. **CTA** - Call to action to launch the app
8. **Footer** - Links and contact information

## Design

- Dark theme matching the Momentum app
- Lavender and gold accent colors
- Responsive grid layouts
- Smooth animations and transitions
- Mobile-first responsive design

## Features

- Smooth scroll navigation
- Intersection observer for scroll animations
- Hover effects on cards
- Fully responsive on mobile, tablet, and desktop
- Accessibility-friendly semantic HTML

## Deployment

To deploy this website:

1. Upload all files to your web server
2. Update links in the footer (transparency, privacy policy)
3. Update the app launch link to point to your production app
4. Configure email for contact form (if adding one)

## Customization

- Update colors in `styles.css` `:root` section
- Modify content in `index.html` sections
- Add more features or research focus areas as needed
- Update footer links and contact information

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
